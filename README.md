# Exotica Agency Website

Modern marketing website for Exotica Agency — brand activation, digital marketing, and market research in Mauritius.

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:5174

## Build

```bash
npm run build
npm run preview
```

## Tech Stack

- React 18 + TypeScript + Vite
- Tailwind CSS + Framer Motion
- Responsive, single-page design

## Sections

- Hero
- About & Values
- Services (Brand Activation, Digital Marketing, Market Research, 360 Campaigns)
- Brand Intelligence (Brand Pulse platform showcase)
- Our Approach (6-phase timeline)
- Team
- Contact
